面向数组的编程

面向数组的编程是以向量化代替使用繁杂的Loop循环。
下面是一个例子，我们要对一个普通网格的数值计算$sqrt(x^2 + y^2)$

meshgrid 的使用方法：
meshgrid函数用两个坐标轴上的点在平面上画格。[X,Y] = meshgrid(x,y) 将向量x和y定义的区域转换成矩阵X和Y，这两个矩阵可以用来表示mesh和surf的三维空间点以及两个变量的赋值。其中矩阵X的行向量是向量x的简单复制，而矩阵Y的列向量是向量y的简单复制。
